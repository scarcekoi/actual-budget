## Usage

1. Copy the CSS of your preferred flavor and accent combination from [themes](themes).
2. Open the app and go to **Settings** > **Show advanced settings** > **Experimental features**.
3. Click **I understand the risks, show experimental features**.
4. Enable **Custom themes**.
5. Scroll up to the **Themes** section, open the dropdown, and select **Custom theme**.
6. Paste the CSS into the provided box.
7. Enjoy!
